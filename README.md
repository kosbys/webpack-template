 Webpack-Template </h1>
My template to get started quickly on projects

# :blue_book: Get started

```bash
git clone git@github.com:kosbys/webpack-template.git
cd webpack-template
npm install
```

##  :wrench: Tools

- Webpack
- Babel
- TypeScript
- Sass
- ESLint + Prettier
- Environment variables with dotenv-webpack
